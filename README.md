💻FZ LAPTOPS-Premium Tech Landing Page

Codsoft Web Development Internship-Task3

*FZ Laptops is a high-end, responsive landing page designed to showcase the latest in portable computing technology. This project was developed as part of the CodSoft Web Development Internship to demonstrate expertise in modern web design principles, including layout structure, aesthetic styling, and cross-device compatibility.

*The page is designed to drive user engagement through a clean, "Apple-style" minimalist interface, high-quality imagery, and clear calls to action.

✨Project Features

*Hero Section: A visually striking introduction with a glassmorphic header and a bold value proposition.

*Product Showcase: Detailed cards highlighting laptop specifications (Processor, RAM, Display) using modern CSS effects.

*Responsive Layout: Fully optimized for Mobile, Tablet, and Desktop using CSS Flexbox and Grid.

*Interactive Navigation: Smooth-scrolling links and hover-animated buttons for an intuitive user journey.

*Modern UI/UX: Features a dark-themed aesthetic with neon indigo accents to match the "Tech" vibe.

🛠Tech Stacks:

*HTML

*CSS
